# Observer Pack 1.0

Observer Pack 1.0 is the first real-world discovery foundation for Bend Score.

The pack is designed to keep discovery terminal-first, offline-first where possible, and respectful of public APIs and source terms. No paid APIs or private scraping are required.

## Planned Observers

- GitHub
- Product Hunt
- Hacker News
- Reddit
- Google Trends
- WordPress Plugins
- Chrome Extensions

## V7 Scope

Only GitHub improvements are implemented in V7.

The GitHub observer uses the public GitHub REST API, supports optional `GITHUB_TOKEN`, and continues with public rate limits when no token is present.

GitHub signals should identify:

- fast-growing repositories
- abandoned popular repositories
- repos with high issue demand
- AI, automation, and tooling projects
- projects with no website or homepage
- commercial wrapper potential
- founder-profile matches

Strong GitHub signals are converted into Bend Score opportunities so they can receive Bend Score, Founder Score, Portfolio Fit, complexity estimates, recommendations, blueprints, memory, and Morning Content signal exports.
