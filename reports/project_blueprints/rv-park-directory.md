## Elevator Pitch

A focused affiliate intelligence site that helps buyers compare options and surfaces high-intent offers from RV Park Directory.

## Target Audience

People actively searching for affiliate site solutions and alternatives.

## Revenue Model

- preserve current revenue first
- affiliate commissions
- sponsorships and newsletter placements

## Suggested Tech Stack

- Start from existing stack: WordPress custom post types
- Add a small Next.js or Python admin layer only if it accelerates content, scoring, or workflow automation.
- Keep analytics, email capture, and affiliate link tracking simple in V1.

## Suggested APIs

- No required API for MVP; start with owned data and manual curation

## Database

PostgreSQL or SQLite for structured location, category, page, and affiliate records.

## Automation Opportunities

- Generate recurring opportunity briefs for Morning Content.
- Track stale pages, missing affiliate links, and conversion gaps.
- Generate page briefs by city, category, route, or comparison cluster.
